### How to generate `docs/preview.png`

1. Open `roadmap_dashboard.html` in your browser.
2. Ensure `js/config.js` contains **generic sample data** (no confidential information).
3. Click **Export Full Roadmap**.
4. Rename the downloaded PNG to `preview.png` and place it in the `docs/` folder:

```
docs/preview.png
```

5. Commit the file so it appears in the repository README.
